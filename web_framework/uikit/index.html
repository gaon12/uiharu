CSS

Uikit 버전은 업데이트 되고, 기존 버전 또한 그대로 사용할 수 있어요!


<link href='https://uiharu.dev/cdn/uikit/3.15.24/css/uikit.min.css' rel='stylesheet' />

혹시 몰라 RTL(Right to Left) 버전도 추가했어요!


<link href='https://uiharu.dev/cdn/uikit/3.15.24/css/uikit-rtl.min.css' rel='stylesheet' />
